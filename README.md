## 项目运行

```
git clone git@gitlab.zudeapp.com:qckj/qc_bussiness_h5.git

cd qc_bussiness_h5

# 全局安装taro脚手架
npm install -g @tarojs/cli

# 安装项目依赖
npm install

# 支付宝小程序开发环境编译
npm run dev:alipay

# 支付宝小程序线上环境编译
npm run build:alipay

# pages模版快速生成 
npm run tep `文件名`

```
